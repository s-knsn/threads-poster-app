初期設定

GitHubのアクセスキー(Fine-grained PAT)をこの端末に保存します。キーは端末の外に送信されません(GitHub APIとの通信のみに使用)。

  1. GitHub → Fine-grained token発行ページ
  2. Repository access: Only select repositories → threads-poster
  3. Permissions: Contents = Read and write
  4. 生成されたキー(github_pat_…)を下に貼り付け

🔒 スマホを紛失した場合は、GitHub側でこのPATを削除(Revoke)すれば即座に無効化できます。
